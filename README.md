## ⚡ projectZero

**projectZero** is a lightweight, modular wireless pentesting platform designed for real-world testing scenarios.

Built for all who want full control over their WiFi environment.

---

## 🔥 Core Capabilities

> **Hybrid Wi-Fi Offensive & Research Framework**  
> Evil Twin • WPA3 SAE Overflow • Wardriving • Sniffer • Karma  
> Powered by **ESP32-C5** | UI via **CardputerADV**


- 📡 Real-time WiFi scanning  
- ⚔️ Active & passive attack modules  
- 🧠 Efficient, low-overhead operation  
- 🔌 Expandable hardware ecosystem  
- 🎛️ External control (Flipper / M5Stack devices / UART / UI)
